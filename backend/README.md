# Fire detector

Backend para un detector de incendios WiFi que envía correos y manda notificaciones por telegram.

# Tecnologías

- NodeJS
- Nodemailer
- MQTT
- Winston
- Babel

# Herramientas adicionales

- MQTTBox
- POSTMAN

# Servicios utilizados

- Heroku
- EMQX Cloud (Broker MQTT) [Link](https://cloud-intl.emqx.com/)
